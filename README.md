    # Blockchain & Cybersecurity Toolkit (Starter)

This is a teaching/demo project with both backend and frontend.

## Backend
    cd backend
    copy .env.example to .env and fill MONGO_URI and JWT_SECRET
    npm install
    npm run dev

## Frontend
    cd frontend
    npm install
    npm run dev

Frontend expects backend at http://localhost:4000 by default. Set VITE_API_URL if different.
